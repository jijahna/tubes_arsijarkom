Materi Router





















ROUTER :

Router adalah suatu perangkat yang digunakan untuk mentransmisi paket data internet melalui proses routing pada perangkat lain. Proses tersebut merupakan proses untuk meneruskan satu paket jaringan dengan paket jaringan lainnya. Yaitu perangkat yang bisa menghubungkan antara dua jaringan atau lebih, perangkat tersebut bertugas untuk mengelola jalannya suatu jaringan dan meneruskan paket data pada alamat IP yang nantinya beberapa perangkat dapat terhubung pada koneksi internet secara bersamaan.

Cara Kerja Router :

Yaitu dengan menggunakan routing table untuk mengarahkan jaringan data agar dapat menentukan jalur yang dapat dilalui oleh sebuah paket sehingga bisa mencapai tujuan. Sebelum melakukan pengiriman paket data maka akan terlebih dahulu membaca header pada aliran paket, selain itu perangkat ini juga bekerja untuk menganalisis arus yang paling efisien agar tujuan dapat terjangkau lalu paket data jaringan akan dikirimkan pada perangkat yang sudah ditentukan.


Fungsi Router :

1. Menghubungkan Jaringan ke DSL

Menghubungkan Jaringan ke DSL Fungsi yang pertama adalah sebagai alat penghubung jaringan lokal pada koneksi DSL (Digital Subscribe Line), biasanya juga berfungsi untuk melindungi data atau firewall sehingga dapat meningkatkan cyber security yang kemudian membuat jaringan internet menjadi lebih aman. Dengan meningkatnya cyber security maka perangkat tersebut dapat memblokir lalu lintas data yang tidak dikenal dan untuk membuat kinerja jaringan tidak lambat karena adanya broadcast storm.

2. Mentransmisikan Informasi

Fungsi lainnya adalah untuk mentransmisikan informasi pada jaringan lain, sistem kerja dari perute yaitu networking bridge yang bisa menjadi jembatan bagi akses internet agar terhubung dan bisa dipakai oleh jaringan lainnya. Hal tersebut juga membuat komputer bisa saling bertukar data dan berkomunikasi dalam suatu jaringan.

3. Membaca Alamat IP

Perangkat ini juga mampu membaca sumber dan juga tujuan dari alamat IP sehingga menentukan proses routing dalam jaringan dari satu node ke node yang lainnya.

4. Menyaring Paket Data

Fungsi dari perangkat ini selanjutnya adalah untuk memfiltrasi paket data yang ada pada suatu jaringan, proses filtrasi tersebut dapat mendukung kinerja dari suatu jaringan sehingga tidak ada perlambatan pada koneksi yang diakibatkan oleh traffic yang besar. Proses penyaringan tersebut dilakukan sesuai dengan aturan alamat IP, port TCP, dan UDP.

5. Menghubungkan Jaringan

Untuk menjalankan fungsi utama perangkat ini yaitu menghubungkan jaringan internet pada perangkat yang tersedia, proses tersebut berlangsung karena adanya pembagian alamat IP pada perangkat jaringan tertentu.


Jenis-jenis Router :

  • Wireless
  • Virtual Router
  • Core Router
  • Edge Router
